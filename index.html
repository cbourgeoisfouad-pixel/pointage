Détail

Touche une case pour fixer ou modifier l'horaire d'un jour (utile pour préparer la semaine suivante). Par défaut, c'est la semaine type qui s'affiche ; les cases modifiées apparaissent en bleu. Pour changer les horaires habituels qui se répètent, va dans « Semaine de référence » (ou page « Salariés » → 🕗 Horaires).

Semaine de référence

Horaires de base appliqués automatiquement à toutes les semaines à venir. C'est la trame qui se répète.

Touche une case pour régler l'horaire habituel d'un jour. Pour un changement ponctuel sur une semaine donnée, utilise « Plannings semaine ».
